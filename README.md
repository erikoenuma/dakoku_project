## アプリ概要
今回のテーマ: 「打刻管理」 <br>
開発期間約1ヶ月 <br>

業務委託契約を結んでいる方々向けの打刻管理アプリです。詳しくはカタログ設計を見てください。

## 環境
* Ruby 3.0.1
* Rails 6.0.5

## 実行手順
1. 任意のディレクトリにgit cloneする
```
$ git clone git@github.com:erikoenuma/dakoku_project.git
```
2. ディレクトリに移動
```
$ cd dakoku_project
```
3. 以下のコマンドを実行
```
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails db:seed
```
4. サーバーを起動
```
$ rails s
```
5. 以下のURLにアクセスする
```
$ http://localhost:3000/
```
