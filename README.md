# カタログ課題2「フリーランス打刻」

## 環境
* Ruby 3.0.1
* Rails 6.0.5

## 実行手順
1. 任意のディレクトリにgit cloneする
```
$ git clone git@github.com:erikoenuma/freelance_dakoku.git
```
2. ディレクトリに移動
```
$ cd freelance_dakoku
```
3. 以下のコマンドを実行
```
$ bundle install
$ rails db:create
$ rails db:migrate
```
4. サーバーを起動
```
$ rails s
```
5. 以下のURLにアクセスする
```
$ http://localhost:3000/
```
